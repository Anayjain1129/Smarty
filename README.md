# Smarty
A tool with which you can just do gestures in the air which gets registered as volume down, up or skip song, or do anti clock with just one finger to do more stuff!
I tried to make this during fallout! and the firmware was written during fallout although this is just a barebone thing cause writing the firmware without physical hardware is not really possible! 

<img width="1142" height="829" alt="image" src="https://github.com/user-attachments/assets/a0082979-4c80-4996-bc3d-476bf923f389" />

Features: 
- control music without using mouse or keyboard
- switch between tabs
- switch between windows
- close windows
- open spotify(or any other app)
- uses ESP32-C3
